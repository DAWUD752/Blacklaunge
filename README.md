# Warkop BlackLounge Landing Page

Landing page statis untuk Warkop BlackLounge, menampilkan informasi lokasi, menu, galeri, dan suasana warkop 24 jam.

## Tech Stack
- HTML5
- CSS3 (Vanilla)
- JavaScript (Vanilla)

## Fitur
- Responsive Design (Mobile First)
- Smooth Scrolling
- Sticky Navbar dengan Glassmorphism
- Animasi Fade-in (Intersection Observer)
- Lightbox Gallery
- Menu Tab Filter
